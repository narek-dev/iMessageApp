# Instalation

NPM

```bash
npm install
```

Yarn

```bash
yarn install
```

# Usage

## For android


```bash
npm Start
```

in another terminal

```bash
npm Android
```

## For iOS

For ios at first need to install pods in the ios directory

```bash
cd ios/
pod install
```

After pods installation go to the root folder

```bash
cd ../
npm run start
```

and in another bash run this

```bash
npm run ios
```
